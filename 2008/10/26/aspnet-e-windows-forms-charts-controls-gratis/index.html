ASP.NET e Windows Forms Charts Controls gratis

October 26, 2008
asp.net charts controls microsoft visual studio windows forms

Foram disponibilizados o controlo de gráficos para ASP.NET e Windows Forms pela equipa de SQL Server Reporting Services e pelo grupo de Data Visualization gratuitamente.

Podem ser usados no Visual Studio 2008, e são bastante identicos aos disponibilizados no SQL Server Reporting Services 2008.

O download para a .NET Framework 3.5 SP1 está disponivel aqui. Podem ainda fazer o download do addon aqui, que adiciona o controlo à toolbox e ao intellisence do Visual Studio 2008.
